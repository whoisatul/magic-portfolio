## Overview

NeuroVisionAI is a deep learning-powered medical imaging platform designed to perform **automatic brain tumor segmentation from MRI scans**. The system uses a **U-Net convolutional neural network trained on the BraTS 2021 dataset** to identify tumor regions at the pixel level, enabling accurate measurement of tumor boundaries and progression.

To make this technology accessible, the deep learning pipeline is integrated into a **full-stack web application** that allows users to upload MRI images and visualize tumor segmentation results instantly. The platform combines **medical imaging AI with an interactive web interface**, enabling real-time predictions and visual overlays that highlight tumor regions.

This project demonstrates the integration of **deep learning research, medical image processing, and modern web technologies** into a deployable AI application.

## Key Features

- **Automated Tumor Segmentation**  
  Uses a trained U-Net model to identify tumor pixels in MRI scans and generate precise segmentation masks.

- **Real-Time Prediction Interface**  
  Users can upload MRI slices and instantly visualize segmentation overlays generated by the deep learning model.

- **High-Accuracy Medical AI Model**  
  Achieves strong segmentation performance using a hybrid **Dice Loss + Binary Crossentropy loss function** optimized for highly imbalanced medical datasets.

- **Interactive Visualization**  
  Displays prediction outputs and segmentation masks through an intuitive web interface, helping users easily interpret model predictions.

- **Efficient Lightweight Architecture**  
  The optimized U-Net architecture balances accuracy and performance, enabling fast inference even on limited hardware.

## Dataset

The model was trained using the **RSNA-ASNR-MICCAI BraTS 2021 dataset**, a widely used benchmark for brain tumor segmentation research.

- Utilized **FLAIR MRI modality**, which highlights tumor-related edema by suppressing cerebrospinal fluid signals.
- Dataset contains **1,251 patient cases** producing **48,000+ MRI slices**.
- Non-informative slices were filtered to improve training quality and reduce dataset bias.

## Model Architecture

The system implements a **lightweight U-Net convolutional neural network** designed for high segmentation accuracy with efficient computation.

- **Input Shape:** 128×128 grayscale MRI slices  
- **Encoder:** Progressive feature extraction layers with filters 32 → 64 → 128  
- **Bottleneck:** Dense feature representation with 256 filters  
- **Decoder:** Upsampling layers with skip connections restoring spatial information  
- **Output:** Sigmoid activation generating binary tumor segmentation masks

The architecture contains approximately **7.8 million parameters**, allowing fast inference while preserving spatial accuracy.

## Challenges and Learnings

One of the main challenges was addressing the **extreme class imbalance in medical imaging datasets**, where tumor regions occupy only a small percentage of total pixels. To overcome this, a hybrid **Dice Loss and Binary Crossentropy loss function** was implemented, significantly improving segmentation accuracy.

Another key challenge involved **preprocessing complex MRI volumes**, including normalization, slice extraction, and resizing while preserving clinically relevant information.

This project also provided valuable experience in **deploying deep learning models within a full-stack application**, connecting a TensorFlow inference backend with a modern Next.js frontend.

## Outcome

After training for 30 epochs on GPU infrastructure, the model achieved a **validation Dice score of 93.74%**, demonstrating strong performance in detecting tumor boundaries and segmentation regions.

NeuroVisionAI showcases how **deep learning, medical imaging, and modern web development** can be combined to create an interactive AI system capable of assisting with complex medical image analysis.
